# Project Summary

This repository is a portfolio summary of selected Optimization for Data Science coursework.

The focus is on understanding and applying optimization methods commonly used in data science and machine learning.

## Main Focus Areas

- Gradient-based optimization
- Step size selection
- Neural network optimization
- Accelerated gradient methods
- Visualization of convergence behavior

## Learning Outcomes

- Implemented and analyzed optimization algorithms in Python.
- Studied how step size affects convergence and stability.
- Compared standard and accelerated optimization approaches.
- Visualized optimization behavior to better understand algorithm performance.

## Tools Used

Python, NumPy, Matplotlib, Jupyter Notebook
